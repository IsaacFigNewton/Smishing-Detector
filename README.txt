TFDIF example (mainly inspiration):	https://www.youtube.com/watch?v=vZAXpvHhQow

Using SBERT because it's the most effective open source embedding model I could find
SBERT Sentence Embedder:		https://sbert.net/

To use this:
	1 Download all the files
	2 Do "pip install <library name>" for the following libraries:
		numpy
		matplotlib
		nltk
		pandas
		scikit-learn
	3 Open main.py
	4 In main.py, uncomment the following lines:
		"import nltk" at the top of main.py
		"nltk.download()" at the top of the main function
	5 Run it
	6 In the GUI, select to download all the packages
	7 Click "Download" and wait
	8 Close the GUI
	9 Pray
	10 Observe the pretty numbers
